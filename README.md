# Site da Disciplina SI401 – Programação para a Web

O site da disciplina **SI401 – Programação para a Web** é uma versão preliminar desenvolvida como parte da **Atividade de Fixação das Unidades 02 a 08**.  
O projeto apresenta os principais aspectos da disciplina, incluindo ementa, objetivos, tópicos abordados, critérios de avaliação e referências, utilizando **HTML e CSS** validados conforme os padrões do **W3C**.

## Funcionalidades

- Página inicial com introdução à disciplina  
- Seções de ementa, objetivos e metodologia  
- Lista de tópicos e conteúdos programáticos  
- Critérios de avaliação e referências bibliográficas  
- Inclusão de imagens e links externos  
- Layout estruturado com elementos semânticos do HTML5  
- Formatação com folhas de estilo externas (CSS)

## Tecnologias e Skills Aplicadas

- **Linguagens:** HTML5, CSS3  
- **Validação:** Padrões W3C (HTML e CSS)  
- **Layout:** Estruturação semântica com cabeçalho, seções e rodapé  
- **Boas práticas:** Separação entre conteúdo e estilo, design responsivo, código limpo e organizado
